Web Crypto

The reality is, risk is complex, and it's about practical ways to reduce surface area. Web crypto is an additional layer that makes it much harder for an attacker to reach someone's private data.

Self hosting is yet another way, Open source is another (allows verification),Signed javascripts are another, and so on.

Just because even combined it's not perfect, doesn't mean users shouldn't have access to the best thing available instead of using file sharing programs with none of these features
